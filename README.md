# Диплом ИИ — генератор студенческих работ по ГОСТ

Веб-сервис (аналог marsik.ai): пользователь вводит тему — получает готовую работу
(ВКР, курсовую, реферат, доклад) на русском языке, оформленную по ГОСТ, со встроенным
редактором и экспортом в Word и PDF. Движок генерации — DeepSeek V4 Flash через OpenRouter.

## Возможности

- Генерация структуры (содержания): тема, объект, предмет, цель, задачи, главы и параграфы.
- Полный текст: введение, главы с выводами, заключение, список источников.
- Оформление по ГОСТ: Times New Roman 14, интервал 1,5, поля 30/10/20/20 мм, нумерация,
  титульный лист, лист задания (для ВКР), автособираемое содержание (поле Word), таблицы.
- Список источников по ГОСТ Р 7.0.100–2018 (учебники ЭБС Юрайт/Литрес, законы, статьи, статистика).
- Текст без markdown-разметки и служебных символов.
- Встроенный WYSIWYG-редактор и экспорт в DOCX / PDF (PDF включает титул, задание, содержание).
- Шаблоны титульных листов: Синергия, ВШЭ, РАНХиГС, МГУ, РУДН, стандарт ГОСТ.
- Увеличенный объём ВКР: параграфы ~2800 слов (7–8 стр.), генерация частями.

## Запуск

```bash
npm install
npm start
```

Откройте http://localhost:3210

## Настройка (`.env`)

```
OPENROUTER_API_KEY=...        # ключ OpenRouter
OPENROUTER_MODEL=deepseek/deepseek-v4-flash
PORT=3210
```

## Структура

- `server.js` — Express-сервер и API.
- `src/openrouter.js` — клиент OpenRouter.
- `src/generator.js` — конвейер генерации (план, разделы, источники).
- `src/docx.js`, `src/pdf.js`, `src/fullDocument.js`, `src/templates.js` — экспорт и шаблоны.
- `src/textutil.js`, `src/blocksToHtml.js`, `src/htmlToBlocks.js` — обработка текста.
- `public/` — премиальный интерфейс (лендинг, редактор).

## API

- `GET  /api/types` — типы работ.
- `POST /api/outline` `{ topic, workType }` — план работы.
- `POST /api/generate` `{ topic, workType, outline, meta }` — потоковая генерация (NDJSON).
- `POST /api/export/docx` `{ html, meta, cfg, outline }` — экспорт в Word.
- `POST /api/export/pdf` `{ html, meta, cfg, outline }` — PDF с титулом, заданием и содержанием.
- `POST /api/export/preview` — полный HTML для предпросмотра.
- `GET  /api/templates` — шаблоны титульных листов вузов.
