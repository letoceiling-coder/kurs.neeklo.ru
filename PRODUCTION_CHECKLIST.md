## Требование 1: Техническое оформление

### ✅ Форматирование текста
- **Шрифт**: Times New Roman 14 pt (`src/docx.js` SIZE_MAIN=28, 14pt в half-points)
- **Красная строка**: 1.5 см (`FIRST_LINE = convertMillimetersToTwip(15)`)
- **Интервал**: 1.5 (`LINE_15 = 360` half-points)
- **Выравнивание**: По ширине (`AlignmentType.JUSTIFIED`)

### ✅ Поля документа
- Левое: 30 мм
- Правое: 10 мм
- Верхнее: 20 мм
- Нижнее: 20 мм
(Установлены в `src/docx.js` функция `exportDocx`)

### ✅ Объем работы
- **Минимум**: 60 страниц (~18,000 слов)
- **Максимум**: 80 страниц (~24,000 слов)
- **Расчёт**: 300 слов ≈ 1 страница (Times New Roman 14, интервал 1.5)
- **Настройка**: `wordsPerSubsection: 2600` для педагогики = ~25 страниц на главу × 3 = 75 стр.

## Production Improvements

### 1️⃣ Санитизация текста
- **Функция**: `sanitizeAllBlocks()` в `productionValidation.js`
- **Убирает**: `КОНЕЦ ТАБЛИЦЫ`, двойные запятые `,,`, двойные точки `..`, символы без ширины
- **Результат**: Чистый текст без артефактов

### 2️⃣ Таблицы в GOST формате
- **Исправления** в `src/docx.js`:
  - Заголовок **жирный**, фон серый
  - Поля ячеек: 100 twip (не давит на края)
  - Ширина таблицы: 95% (оставляет поля)
  - Интервал между таблицей и текстом: 120 twip (безопасный отступ)

### 3️⃣ Валидация структуры
- **Функции** в `productionValidation.js`:
  - `validateVkrStructure()` — проверка наличия всех секций
  - `validateChapterBalance()` — проверка баланса ±15%
  - `validateIntro()` — проверка всех 10 пунктов введения
  - `validateTable()` — проверка каждой таблицы на GOST

### 4️⃣ Автоисправления в `generator.js`
- Все блоки санитизируются перед аудитом (`autoFixBlocks()`)
- Введение дополняется недостающими пунктами (`ensureIntroStructure()`)
- Глав увеличены до 2600 слов/раздел для баланса

## API для проверок

```bash
# На сервере:
cd /var/www/kurs.neeklo.ru

# Полная проверка документа
node scripts/compliance-check.mjs <docId>

# Сводка блоков
node scripts/doc-blocks.mjs summary <docId>

# Поиск
node scripts/doc-blocks.mjs find <docId> --kind p --pattern "актуальн"

# Исправить markdown-таблицы (если остались)
node scripts/doc-blocks.mjs fix-leaks <docId>
```
